Typography

  • Clear hierarchy: heading ‚Üí subheading ‚Üí body ‚Üí caption
  • Line-height 1.4‚Äì1.6 for body, 1.1‚Äì1.2 for headings

Color System
  • Min 4.5:1 contrast ratio (WCAG AA)
  • All colors as CSS custom properties
  • Semantic naming: --color-surface, --color-danger
  • Never hardcode hex values in components
‚ñ¶
Spacing
  • 4px base unit ‚Äî all spacing is a multiple of 4
  • Scale: 4, 8, 12, 16, 24, 32, 48, 64, 96
  • Consistent padding within component families
  • No magic numbers ‚Äî ever
‚óà
Motion & Interaction
  • Micro-interactions: 150ms ease-out
  • Page transitions: 250‚Äì350ms
  • Never animate without purpose
  • Always respect prefers-reduced-motion
